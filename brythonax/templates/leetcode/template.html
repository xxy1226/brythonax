    2.

      2. 
                    
        
                        
      1. YES: add it to the final list arr.
        2. 
                    
      
                    
    2. NO: if a valid +k or -k digit can be generated to the end of the current number, pass the new number into the recursion again.

      2. 
                    
        
                        
      1. 是：将其加入最终列表arr中。
        2. 
                    
      
                    
    2. 不是：如果可以在末尾生成合法的 +k 或 -k 数，则将新的数字再次传入递归。
